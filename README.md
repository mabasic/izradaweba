# Izrada weba

## Quickstart

```
npm install

# to compile and run once
sbtn run

# or to compile and run on source change
sbtn ~reStart
```
Other useful commands:

```
# run this on work start (sbt server)
sbtn

sbtn reStatus
```

## Scala.js

```bash
# development
sbt js/fastLinkJS

# production
sbt js/fullLinkJS
```

## Tailwind CSS

```bash
# watch
npm run watch

# production
npm run prod
```
